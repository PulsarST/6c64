## overview
Эта команда ==6c64==
Наша команда создала сюжетную игру

Главный герой хочет пригласить на балл свою возлюбленную.
Для этого он хочет купить цветы и получить машину
Для этого он побеждает своего друга в танцевальной битве и собирает деньги с доставки на цветы

Помогите ему в этой непростой миссии !

## Key words
1. Медеу сити (город, 2 уровень)
2. Кибер-юрта (город состоит из юрт, 2 уровень)
3. Эко-дирижабль
4. Наноимплант

## Building

> **requirements:**
> 1. cmake
> 2. Ninja

``` powershell
cmake -G Ninja -S . -B build
cd build
Ninja

strip PunkVerseProject.exe
PunkVerseProject
pause
```
