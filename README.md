## overview
Эта команда ==6c64==
Наша команда создала сюжетную игру

Главный герой хочет пригласить на балл свою возлюбленую.
Для этого он хочет купить цветы и получить машину
Для этого он побеждает своего друга в танцевальной битве и соберает деньги с доствки на цветы

Помогите ему в этой непростой миссии !

## Key words
1. Медеу сити (город 2 уровень)
2. кибер юрта (город состоит из юрт2 уровень)
3. эко дерижабль
4. наноимплант

## Building

> **requirements:**
> 1. cmake
> 2. Ninja

``` powershell
cmake -G Ninja -S . -B build
cd build
Ninja

strip PunkVerseProject.exe
PunkVerseProject
pause
```
